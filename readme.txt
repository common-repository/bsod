== Description ==

BSoD for WordPress is a replacement for your 404 WordPress pages.
It will show a BSoD when a visitor get an 404 Request Response.
NB: This plugin was developed to honor the partnership between Microsoft and WordPress :)

== Installation ==

This section describes how to install the plugin and get it working.
e.g.
1. Upload `bsod` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
